## Practice NestJS TypeScript

### Installation Guide

#### 1. Install npm packages

```bash
npm install
```

#### 2. Start application

```bash
npm start
```

#### 3. Run Redis server (optional)

```bash
redis-server
```

#### 4. Run by using Docker (optional)

```bash
docker-compose build
```

```bash
docker-compose up -d
```
